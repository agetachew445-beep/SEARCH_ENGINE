        READ THIS FIRST!
========================================

You are seeing "127.0.0.1 refused to connect" because:

❌ Python is NOT installed on your computer
❌ The Django server is NOT running

========================================
  QUICK FIX (3 Steps)
========================================

STEP 1: Install Python
-----------------------
Go to: https://www.python.org/downloads/
Download and install Python 3.11
⚠️ CHECK THE BOX: "Add Python to PATH"

OR

Open Microsoft Store → Search "Python 3.11" → Install

STEP 2: Run Installation
-------------------------
Double-click: check_and_install.bat
Follow the prompts

STEP 3: Start Server
--------------------
Double-click: start_server.bat
Open browser: http://127.0.0.1:8000/

========================================
  THAT'S IT!
========================================
