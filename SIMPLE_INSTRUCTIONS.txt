═══════════════════════════════════════════════════════════════════
  MANUAL PYTHON INSTALLATION (if Microsoft Store doesn't work)
═══════════════════════════════════════════════════════════════════

   1. Go to: https://www.python.org/downloads/
   2. Download Python 3.11
   3. Run installer
   4. CHECK THE BOX: "Add Python to PATH"
   5. Click "Install Now"
   6. Run CLICK_ME_TO_INSTALL.bat
