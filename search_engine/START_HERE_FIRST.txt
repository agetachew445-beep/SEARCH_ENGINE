┃  STEP 1: Install Python (Choose ONE option)                   ┃
┗━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━┛

   OPTION A: From Python.org (Recommended)
   ────────────────────────────────────────
   1. Open your web browser
   2. Go to: https://www.python.org/downloads/
   3. Click "Download Python 3.11" (big yellow button)
   4. Run the installer
   5. ⚠️  CHECK THE BOX: "Add Python to PATH" ⚠️
   6. Click "Install Now"
   7. Wait 2-3 minutes
   8. Done!

   OPTION B: From Microsoft Store (Easier)
   ────────────────────────────────────────
   1. Press Windows key on keyboard
   2. Type "Microsoft Store"
   3. Open Microsoft Store
   4. Search "Python 3.11"
   5. Click "Install"
   6. Wait 2-3 minutes
   7. Done!

═══════════════════════════════════════════════════════════════════
  TROUBLESHOOTING
═══════════════════════════════════════════════════════════════════

   Problem: "Python not found"
   ───────────────────────────
   → Install Python (see STEP 1)
   → Make sure "Add to PATH" was checked

   Problem: "check_and_install.bat doesn't work"
   ──────────────────────────────────────────────
   → Install Python first
   → Close all Command Prompt windows
   → Try again

   Problem: "Port already in use"
   ──────────────────────────────
   → Close other programs
   → Or edit start_server.bat
   → Change "runserver" to "runserver 8080"
